# 📚 Library Management System (Python)

A simple **Library Management System** built using **Python and Object-Oriented Programming (OOP)**.
This project allows users to manage books and students in a library environment through a **menu-driven command line interface (CLI)**.

---

## 📌 Project Type

Command Line Interface (CLI) application for managing basic library operations.

---

## 🚀 Features

* Add new books to the library
* Register new students
* Issue books to students
* Return books
* Display all available books

---

## 🧠 Concepts Used

This project demonstrates the following Python concepts:

* Python **Classes and Objects**
* **Constructors (`__init__`)**
* **Inheritance** (`Student` inherits from `Person`)
* Lists for storing objects
* **Exception Handling (`try-except`)**
* Pythonic searching using **`next()` and generator expressions**
* **Menu-driven CLI program**

---
